# Josephus Problem Solver

An interactive visualization of the Josephus problem built using
Node.js, Express, HTML, CSS, and JavaScript.

## Live Demo
https://josephus-problem-solver.onrender.com/

## Features
- Interactive circle generation
- Step-by-step elimination simulation
- Guess the survivor feature

## Run locally
npm install
npm start
